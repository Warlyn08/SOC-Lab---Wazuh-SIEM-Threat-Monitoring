# SOC-Lab---Wazuh-SIEM-Threat-Monitoring
Laboratorio SOC práctico enfocado en monitoreo de seguridad, detección de amenazas, análisis de eventos, gestión de incidentes y administración de SIEM utilizando Wazuh, Windows 10, Kali Linux, Metasploitable 2 y VMware Workstation. Incluye integración de agentes, escaneo de vulnerabilidades, troubleshooting y operaciones Blue Team.

# Infraestructura Implementada

🛡️ Wazuh Server (SIEM / XDR)

🖥️ Windows 10 SOC-01 (Agente)

🖥️ Windows 10 SOC-02 (Agente)

🐉 Kali Linux (Ataques controlados)

🎯 Metasploitable 2 (Máquina vulnerable)

🌐 VMware Workstation (Virtualización)

# Capacidades del Laboratorio

✔ Monitoreo centralizado de eventos de seguridad

✔ Detección de amenazas en tiempo real

✔ Gestión de agentes Windows y Linux

✔ Análisis de logs del sistema

✔ Simulación de ataques controlados

✔ Evaluación de vulnerabilidades

✔ Correlación de eventos en SIEM
