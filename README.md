# SOC-Lab---Wazuh-SIEM-Threat-Monitoring
Laboratorio SOC práctico enfocado en monitoreo de seguridad, detección de amenazas, análisis de eventos, gestión de incidentes y administración de SIEM utilizando Wazuh, Windows 10, Kali Linux, Metasploitable 2 y VMware Workstation. Incluye integración de agentes, escaneo de vulnerabilidades, troubleshooting y operaciones Blue Team.
